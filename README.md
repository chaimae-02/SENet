# SENet Transfer Learning - Sugarcane Leaf Disease Classification

## Description
Ce projet utilise **SENet154** pour la classification d’images de feuilles de canne à sucre via **transfert learning**.  
L’objectif est de détecter et classifier différentes maladies des feuilles (dont Yellow Leaf Disease) ainsi que les feuilles saines.

## Prérequis
- Python 3.8+
- TensorFlow 2.x
- Keras
- Autres bibliothèques listées dans `Code/requirements.txt`
